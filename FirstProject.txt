Ένας client (π.χ. browser, Postman, frontend εφαρμογή) στέλνει ένα HTTP GET request στο http://localhost:8080/hello.
Το @GetMapping("/hello") αντιστοιχεί το URL /hello στη μέθοδο hello().
Η μέθοδος hello() εκτελείται και επιστρέφει "Hello World".
Το @RestController μετατρέπει αυτόματα την έξοδο σε HTTP Response.

Το @PostMapping("/hello") δηλώνει ότι η μέθοδος helloPost() ανταποκρίνεται σε HTTP POST αιτήματα που στέλνονται στο endpoint /hello.

🔹 Το @RequestBody String name σημαίνει ότι το body του HTTP request θα μετατραπεί σε μια Java String μεταβλητή.

Ένας client (π.χ. Postman, JavaScript Fetch API) στέλνει ένα HTTP POST request στο http://localhost:8080/hello.
Το @RequestBody παίρνει το περιεχόμενο του request body και το αποθηκεύει στη μεταβλητή name.
Η μέθοδος helloPost() επιστρέφει μια απλή συμβολοσειρά (Hello <name>!).
Το Spring Boot μετατρέπει το αποτέλεσμα σε HTTP response

Το @GetMapping("/hello/{name}") δηλώνει ότι αυτή η μέθοδος ανταποκρίνεται σε HTTP GET requests στο URL /hello/{name}.

🔹 Το @PathVariable String name σημαίνει ότι το {name} στο URL θα μετατραπεί σε Java μεταβλητή name και θα περάσει στη μέθοδο.
🔹 Η μέθοδος επιστρέφει ένα αντικείμενο τύπου HelloResponse, το οποίο το Spring Boot μετατρέπει αυτόματα σε JSON.

Ο client (π.χ. browser, Postman, JavaScript Fetch API) στέλνει GET request στο http://localhost:8080/hello/John.
Το Spring Boot εξάγει το "John" από το URL και το περνά στη μέθοδο helloParam().
Δημιουργείται ένα νέο αντικείμενο HelloResponse("Hello John").
Το Spring Boot μετατρέπει το αντικείμενο σε JSON και το επιστρέφει ως HTTP response.
